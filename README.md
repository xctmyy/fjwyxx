# fjwyxx 反戒网瘾学校公益广告Vue UI

## 项目介绍
这是一个基于Vue 3开发的反戒网瘾学校公益广告UI组件库，旨在通过各种形式的广告展示，提高公众对非法戒网瘾学校问题的认识。

## GitHub仓库
[github.com/xctmyy/fjwyxx](https://github.com/xctmyy/fjwyxx)

## 组件列表
- HeaderAntiAddictionAd - 页眉广告组件
- FooterAntiAddictionAd - 页脚广告组件
- AntiInternetAddictionAd - 侧边栏广告组件
- InlineAntiAddictionAd - 页面内广告组件

## 功能特点
- 响应式设计，适配不同设备屏幕
- 支持广告关闭功能
- 集成中央网信办举报渠道
- 包含动画效果增强用户体验

## 技术栈
- Vue 3
- TypeScript
- Vite
